NGINX 常用功能

NGINX 是一款高性能的 Web 服务器和反向代理服务器，常用于构建高性能和可伸缩性的 Web 应用程序。除了作为 Web 服务器和反向代理服务器之外，NGINX 还可以实现许多其他功能。在本文中，我们将介绍 NGINX 的一些常用功能。
